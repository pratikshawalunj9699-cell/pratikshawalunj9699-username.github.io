# Pratiksha Walunj — Portfolio Website

Personal portfolio website for **Pratiksha Walunj**, Full-stack .NET Developer.  
Built with **Python / Flask** for local development and **static HTML** for GitHub Pages hosting.

---

## Features

- Animated orbital badge graphic (ASP.NET Core, Angular, PostgreSQL, Microservices, Azure, AWS, Blazor)
- Typewriter cycling title: *Software Engineer / .NET Developer / Full-Stack Developer*
- Dark / light mode toggle with `localStorage` persistence
- Scroll progress bar & animated stat counters
- Scroll-reveal cards (skills, timeline, education, certifications)
- Contact form with `mailto:` handler
- Responsive design (desktop → tablet → phone → 380 px)
- GitHub Pages static export via `build.py`

## Quick Start (Local Dev)

```bash
# 1. Clone the repo
git clone <repo-url>
cd pratikshawalunj_portfolio

# 2. Create virtual environment
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run Flask dev server
python app.py
# → http://localhost:5000
```

---

## Docker

```bash
# Build image
docker build -t pratikshawalunj-portfolio .

# Run container
docker run -p 5000:5000 pratikshawalunj-portfolio
# → http://localhost:5000

# Using Docker Compose (from devops/ or project root)
docker compose -f devops/docker-compose.yml up --build
```

---

## GitHub Pages Deployment

```bash
# 1. Generate static site
python build.py

# 2. Commit & push the docs/ folder
git add docs/
git commit -m "Build static site"
git push

# 3. In your GitHub repo:
#    Settings → Pages → Source: Deploy from branch → main → /docs → Save
```
